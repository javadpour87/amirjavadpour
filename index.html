Core security domains

Focused on defense that adapts under pressure.

The main research direction is not generic cybersecurity. It is about how security mechanisms should behave in real networked systems when the environment is dynamic, adversarial, distributed, latency-sensitive, and infrastructure-aware.

MT

Moving Target Defense

Dynamic mutation, shuffling, slice isolation, and adaptive reconfiguration to reduce attacker certainty and improve resilience against DDoS and persistent disruption.

CD

Cyber Deception

Deception-aware defense strategies using honeypots, diversion, and analytical models that turn uncertainty into an operational advantage for defenders.

IE

IoT and Edge Security

Protection for distributed IoT and edge systems where security must coexist with efficiency, orchestration constraints, and service continuity.

56

5G / 6G and Service Security

Secure slicing, service mesh hardening, encryption-as-a-service, path verification, and trustworthy control of programmable network infrastructures.

Research philosophy

The best security systems are not only accurate or theoretically strong. They are adaptive, deployment-aware, measurable, and usable in the infrastructures where real attacks happen. This profile is built around that idea.
